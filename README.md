# typeditor

A markdown editor inspired by typecho which is tiny but perfect

### Use

dom:

```html
<div id="typeditor"></div>
```

js:

```javascript
let te = new TypeEditor({
  el: '#typeditor',
})
```

#### get/set value

```JavaScript
te.value()
te.value('念念不忘，必有回响')
```
